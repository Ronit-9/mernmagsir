React apps are made out of components. A component is a piece of the UI (user interface) that has its own logic and appearance. A component can be as small as a button, or as large as an entire page.

Angular

Angular is an open-source, TypeScript-based web application framework maintained by Google and a large developer community. It is designed for building dynamic, scalable, and maintainable single-page applications.

VueJS

Vue is an open-source, progressive JavaScript framework for building user interfaces. Created by Evan You in 2014, Vue was designed as an alternative to heavier frameworks like AngularJS and React.
